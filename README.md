# Adrenaline-Slot-Machine

Adrenaline Slot Machine — это браузерная игра-слот машина, написанная на чистом JavaScript, HTML и CSS.

## 🎰 Описание

Это простая, но динамическая слот-игра с системой ставок, балансом и случайными выигрышами. Игрок запускает вращение барабанов и получает результат на основе совпадений символов.

## ⚙️ Как работает игра

- У игрока есть баланс монет
- Перед каждым спином выбирается ставка
- При нажатии кнопки **SPIN** списывается ставка
- Барабаны начинают вращение
- После остановки определяется результат:
  - ❌ 1–2 совпадения → проигрыш
  - 😐 3 совпадения → 0x (ничего не меняется)
  - 🔥 4 совпадения → +0.5x ставки
  - 💥 5 совпадений → +2x ставки

## 🔊 Звук

В игре есть система звуков вращения:
- Random
- Sound 1
- Sound 2

## 💰 Ставки

Игрок может изменять ставку прямо в интерфейсе. Минимальная ставка — 10 монет.
